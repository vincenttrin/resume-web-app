# Portfolio Images

This directory contains images for the portfolio website.

## Required Images

1. **profile.jpg** - Your professional headshot (recommended: 800x800px or higher)
2. **aws-ccp.png** - AWS Cloud Practitioner badge icon (24x24px)
3. **aws-saa.png** - AWS Solutions Architect badge icon (24x24px)

## Project Thumbnails (Optional)

Place project screenshots in the `projects/` subfolder:
- `projects/grid-game.png` - Grid Puzzle Game screenshot
- `projects/morse-code.png` - Morse Code Decoder screenshot
- `projects/aws-serverless.png` - AWS Architecture diagram
- `projects/reporting-system.png` - Automated Reporting System diagram

Recommended thumbnail size: 600x400px
